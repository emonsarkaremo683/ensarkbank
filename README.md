# EnsarkBank

A modern banking application built with Java, TypeScript, HTML, and SCSS.

## 📋 Overview

EnsarkBank is a full-stack banking platform that combines robust backend services with an intuitive frontend interface. The project leverages modern web technologies to provide secure and efficient banking solutions.

## 🛠️ Technology Stack

- **Backend**: Java (59.5%)
- **Frontend**: TypeScript (19.6%)
- **Markup**: HTML (14.3%)
- **Styling**: SCSS (6.6%)

## 🎯 Features

- Secure user authentication and authorization
- Account management
- Transaction processing
- Real-time banking operations
- Responsive user interface

## 🚀 Getting Started

### Prerequisites

- Java 11+
- Node.js 14+
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/emonsarkaremo683/ensarkbank.git
cd ensarkbank
```

2. Install dependencies:
```bash
npm install
```

3. Build the project:
```bash
# Backend
mvn clean install

# Frontend
npm run build
```

### Running the Application

```bash
# Development mode
npm run dev

# Production mode
npm run start
```
